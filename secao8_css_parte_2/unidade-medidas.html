Unidades de medidas

Unidade de medida "em"

Basicamente 1em é igual ao tamanho da fonte do elemento pai, ou seja, o tamanho da fonte padrão.

Mas se usarmos 2em por exemplo, muda para o dobro do padrão.

Uma coisa que se vê muito é determinar o tamanho da fonte já dentro do elemento html com a porcentagem de 62.5%. Isso faz com que o tamanho padrão da fonte seja de 10px.

Mas porque isso é interessante?

Porque dessa forma, 1em equivale a 10px, se eu quiser 16px, basta usar 1.6em. Se eu quiser 18px, basta usar 1.8em. Prático não?

E seu eu quiser meu H1 com 24px, basta usar 2.4em.

Unidade de medida "rem"

A unidade de medida "rem" funciona de forma semelhante ao "em", porém, ao invés de se basear no tamanho da fonte do elemento pai, ela se baseia no tamanho da fonte do elemento raiz (normalmente o elemento html).

Por exemplo, se o tamanho da fonte padrão do html for 16px, então 1rem será igual a 16px. Se quisermos que o H1 tenha 24px, basta usar 1.5rem.

Como o elemento raíz que é o nosso html está parametrizado com 62.5% o que equivale a 10px, este parágrafo tem que ter 28px, pois criei uma classe chamada vinteoitopx com 2.8rem para este elemento p.

E continuando a aula, estas unidades de medida em e rem, não são usadas apenas com fontes, mas também com larguras, alturas, espaçamentos e outros elementos. Veja acima!

Unidade de medida "porcentagem"

A unidade de medida "porcentagem" é uma unidade relativa ao tamanho do elemento pai. Por exemplo, se o elemento pai tem 100px de largura e o elemento filho tem 50%, então o elemento filho terá 50px de largura.

Essa unidade é muito útil para criar layouts responsivos, onde os elementos se ajustam ao tamanho da tela do dispositivo.
